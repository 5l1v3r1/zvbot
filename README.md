# how to install this tool?
```just type `pip2 install zvbot` in your terminal```

# how to use?
```
zvbot --account USER:PASS --run; or
zvbot --cookie PATH --run; or
zvbot --cookie PATH --save-cookie --run; or
whatever :/
```
